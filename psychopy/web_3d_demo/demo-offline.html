About This Demo

This is a standalone WebGL demonstration that runs entirely in your browser without external dependencies.

  • Uses native WebGL API for 3D rendering
  • No external libraries required
  • Hardware-accelerated graphics
  • Real-time interaction

For Full Features

Open index.html to access the full demo with Pyodide integration for running Python code in the browser.

💡 Usage Instructions

Controls:

  • Start/Stop Rotation: Toggle automatic rotation of the 3D object
  • Change Color: Randomize the colors of the shape
  • Change Shape: Switch between different 3D shapes (cube, pyramid)
  • Reset Scene: Return to initial state

Technical Details:

This demo uses WebGL shaders to render 3D geometry with vertex and fragment shaders. The scene includes perspective projection, model-view transformations, and real-time color updates. All rendering is performed on the GPU for smooth 60 FPS animation.
